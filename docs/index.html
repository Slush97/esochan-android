What you get

  • Full board browsing Catalog, pages, threads, and replies for every 4chan board — native UI, not a WebView wrapper.
  • Inline media gallery Pinch-zoom images and GIFs, play video, swipe through a thread’s attachments without leaving the thread.
  • Offline thread saves Archive a whole thread (posts + images) into the app’s storage. No storage permission theater.
  • Tabs, history, favorites Keep several boards and threads open; jump back to recent and bookmarked threads.
  • Themes that match the boards Yotsuba B, Futaba, Photon, Tomorrow, and friends — including custom theme JSON.
  • Modern Android under the hood OkHttp, Media3, Material Components. Targets Android 15, runs back to Android 7.0.

Prebuilt APK

  1. Open the latest release.
  2. Download esochan-1.0.0.apk (or the versioned APK on that release).
  3. Allow install from your browser/file manager if Android asks.
  4. Open the APK and install. Done.

Updates: install a newer release over the old one (same signing key).

Get v1.0.0

Build from source

Needs JDK 17+, the Android SDK, and the NDK.

# clone and build
git clone https://github.com/Slush97/esochan-android
cd esochan-android
./gradlew assembleDebug

# output
build/outputs/apk/debug/esochan-debug.apk

Release builds need a keystore — see the README.
