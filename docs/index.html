/esochan/

a 4chan client for Android

>4chan, on your phone

Android 7.0+ · GPLv3 · no store, no account

Download APK .apk Source GitHub

Features

A rebuild of Overchan Android with a new HTTP stack, a Material interface, and an inline media viewer. Only 4chan now, since the other boards are dead.

Every board

Browse every 4chan board in a native app: catalog, threads, and replies.

Media that opens inline

Pinch-zoom images, GIFs, and video in a fullscreen gallery. Swipe through a thread's attachments without leaving it.

Save threads offline

Archive a whole thread, posts and images, to read offline later. Saved in the app's own storage, no permissions needed.

Rebuilt for modern Android

OkHttp networking, a Media3 viewer, and an AppCompat/Material UI on a current toolchain. Targets Android 15, runs back to Android 7.

Install or build

Prebuilt APK

Grab the latest debug build from GitHub Releases and sideload it. No store, no account.

Download APK

From source

Needs JDK 17+, the Android SDK, and the NDK.

# clone and build
git clone https://github.com/Slush97/esochan-android
cd esochan-android
./gradlew assembleDebug

# output
build/outputs/apk/debug/esochan-debug.apk
